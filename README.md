Human-in-the-Loop Fairness Auditor for AI Decision Systems

This project builds an AI auditing system that detects, explains, and mitigates bias in machine learning decision models.

Dataset
Adult Income Dataset (UCI Machine Learning Repository)

Project Pipeline

1. Dataset preparation
2. Baseline model development
3. Fairness evaluation
4. Explainability analysis (SHAP)
5. Bias attribution
6. Counterfactual fairness testing
7. Human-in-the-loop review
8. Bias mitigation
9. Automated fairness alerts
10. Bias drift detection

Run the system

python main_pipeline.py

Launch dashboard

streamlit run app/dashboard.py
